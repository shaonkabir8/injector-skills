Show the injector-skills quick-reference card.

| Command | What |
|---|---|
| `/injector-skills` | Activate at default level (full) |
| `/injector-skills lite` | Light compression — ~30% tokens dropped |
| `/injector-skills ultra` | Maximum compression |
| `/injector-skills wenyan[-lite\|-ultra]` | Classical Chinese compression |
| `/injector-skills off` | Deactivate |
| `/injector-skills-commit` | Terse commit message |
| `/injector-skills-review` | One-line review findings |
| `/injector-skills-compress <file>` | Compress a Markdown file |
| `/injector-skills-stats` | Lifetime token-savings |

Natural language also works: "turn on injector-skills", "stop injector-skills", "normal mode".

Code, commits, security warnings: injector-skills drops out automatically.
